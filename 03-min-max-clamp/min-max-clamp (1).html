CSS Min() Example - If the viewport at 50vw is less than 700px, I take the value of 50vw, otherwise, I stay at 700px.

CSS Max() Example - If the viewport at 50vw is greater than 700px, I take whatever 50vw's value is, otherwise, I stay at 700px.

CSS Clamp() Example - If the preferred value of 60vw is smaller than 400px, the div will stop shrinking when it reaches that lower bound. Likewise if 60vw is larger than 600px, the div will stop growing when it reaches that upper bound.
